# HANQYZY / hub

Полная система совместной работы для артиста и менеджера.

---

## Что внутри (7 вкладок)

### 📅 Контент-план
Календарь по месяцам и неделям. Каждая запись: название, платформа, формат, дата выкладки, ссылка на сценарий, прикреплённый файл (видео/фото/текст). Таблица + календарная сетка.

### 📝 Сценарии
Отдельная вкладка для сценариев. Карточки с текстом, привязка к контенту. Полноценный редактор.

### 📊 Неделя
График на текущую неделю — собирает контент, задачи и концерты в одном месте.

### ☑ Задачи
Канбан-доска для задач вне контента. Статусы, приоритеты, назначение, дедлайны, комментарии.

### 💿 Релизы
Страница по каждому релизу: дата, стадия (демо → запись → сведение → мастеринг → готов), обложка, текст песни, BPM, тональность, жанр, заметки.

### 🎤 Концерты
Календарь выступлений: дата, время, площадка, город, гонорар, статус, контактное лицо.

### 🤝 Контакты
Кто какое сотрудничество предложил. Имя, компания, роль, предложение, статус переговоров, контакты.

### Лог активности
Боковая панель — видно кто что поменял и когда. Все действия логируются.

## Запуск

### 1. Firebase (бесплатно)

1. [console.firebase.google.com](https://console.firebase.google.com) → Add project
2. Build → Realtime Database → Create Database → **Start in test mode**
3. Project Settings → Your apps → Web → Register app
4. Скопируй `firebaseConfig` в файл `src/firebase.js`

### 2. Локально

```bash
npm install
npm run dev
```

### 3. Деплой на Vercel

1. Залей на GitHub
2. [vercel.com](https://vercel.com) → Add New Project → выбери репо → Deploy
3. Получи ссылку → отправь менеджеру
